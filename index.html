How it works

  1. 01

    Define scope

    Set your target, pick a scope profile, and choose LLM providers. The governance engine freezes the scope — it cannot be expanded mid-operation.

  2. 02

    Automated recon

    Deterministic recon discovers endpoints, CVEs, and technologies before the LLM starts. JS bundle analysis finds hidden APIs. OSINT and cross-run memory load proven payloads.

  3. 03

    Agent execution

    Sub-agents test in parallel across sandboxed containers. Auto-hooks fire reactive tools on findings — SQLi triggers sqlmap, XSS triggers browser probes. Optional Burp Pro runs alongside for passive scanning.

  4. 04

    Structured output

    JSON and Markdown reports with deduplicated findings, attack graph visualization, reproduction steps, and evidence. Proven payloads promote to muscle memory for future runs.
