Capabilities

Deterministic recon + LLM-driven adaptive exploitation.

Hybrid v2 engine

Deterministic detection handles 14 vulnerability types (injection, file access, auth, client-side, config) via pattern matching and behavioral analysis — zero LLM cost. When findings are confirmed or budget allows, an LLM agent takes over for adaptive multi-step exploitation.

Sandboxed Kali execution

Each assessment runs in its own isolated Kali container. 35 security tools (nmap, sqlmap, nuclei, ffuf, nikto, hydra, dalfox, and more) plus a persistent Playwright browser for multi-session auth testing and DOM analysis.

5-stage command validation

Every shell command passes through allowlist, pattern block, scope check, risk classification, and OPSEC default injection before execution. Immutable scope, credential isolation, output sanitization with secret redaction.

Cross-run memory

Three memory tiers — working, tradecraft, and muscle — promote proven payloads from NEW to LEARNING to PROVEN across runs. A git-backed progress repo tracks every tested intent, blocking duplicates before they consume budget.

How it works

Four phases, deterministic first, LLM when it matters.

  1. 70%

    Recon (deterministic)

    Tech profiling, JS bundle analysis, endpoint discovery, OSINT enrichment, sensitive path probing. Zero LLM cost — fast scanners block recon while deferred scanners (nuclei, Burp) run async.

  2. 10%

    Planning (deterministic)

    CVE mapping via NVD, ExploitDB search, hypothesis generation from collected intelligence. Attack surface prioritization before any LLM involvement.

  3. AUTO

    Exploitation (deterministic)

    14 vulnerability types detected via pattern matching and behavioral analysis — SQL injection, XSS, SSRF, IDOR, SSTI, command injection, LFI, XXE, open redirect, CORS, deserialization, and more.

  4. LLM

    Adaptive exploitation

    Claude Agent SDK with Opus orchestrator and specialized sub-agents. Persistent Playwright browser, multi-session auth testing, attack chaining. Professional PDF reports with ATT&CK/OWASP mapping.
