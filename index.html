How it works

  1. 01

    Define scope

    Set target, scope profile, and providers. The governance engine freezes scope — it cannot expand mid-operation.

  2. 02

    Recon

    Deterministic scanners run before the LLM — endpoint discovery, JS bundle analysis, OSINT enrichment. Fast scanners block; deferred scanners (nuclei, Burp) run async and deliver results via callback.

  3. 03

    Agent execution

    Sub-agents test in parallel. Condition-based hooks fire reactive tools on findings — SQLi triggers sqlmap, XSS triggers browser probes. Coverage-aware dedup prevents duplicates while ensuring untested surface gets covered.

  4. 04

    Output

    JSON and Markdown reports with deduplicated findings, attack graph, and evidence. Proven payloads promote to muscle memory for future runs.
