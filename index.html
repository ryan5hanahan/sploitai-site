Capabilities

Everything runs inside sandboxed containers with immutable scope policies. The agent cannot operate outside the boundaries you define.

Orchestrator + sub-agents

An Opus orchestrator dispatches specialized sub-agents — enumerator, assessor, exploiter, reporter — for parallel vulnerability coverage. Coverage-aware continuation loops keep testing until the attack surface is exhausted or the budget runs out.

Sandboxed Kali tooling

Security tools across recon, web discovery, vuln scanning, injection testing, and exploitation run inside sandboxed Kali containers. OPSEC defaults (rate limits, timeouts, random user-agents) are injected automatically.

Smart recon + OSINT

Deterministic reconnaissance runs before the LLM session — endpoint discovery, CVE scanning, technology fingerprinting, and JS bundle analysis. OSINT enrichment from Shodan, Censys, VirusTotal, NVD, and BuiltWith.

Governance-first security

5-stage command validation pipeline: allowlist, pattern blocking, scope check, risk classification, and default injection. Credentials use opaque references — secrets never appear in LLM messages. Output sanitization redacts 14+ secret patterns.

Cross-run memory

Four-tier memory system: working (current run), short-term (per-target), long-term (global playbooks), and muscle memory that promotes proven payloads through NEW, LEARNING, and PROVEN stages.

Multi-provider LLM routing

Phase-aware 3-tier routing across 6 providers — Anthropic, OpenAI, Gemini, Bedrock, Ollama, LM Studio. Mix providers per tier with per-model cost tracking and budget enforcement.

How it works

  1. 01

    Define scope

    Set your target, pick a scope profile, and choose LLM providers. The governance engine freezes the scope — it cannot be expanded mid-operation.

  2. 02

    Automated recon

    A deterministic recon sweep discovers endpoints and CVEs before the LLM session starts. OSINT sources enrich the target profile. Cross-run memory loads proven payloads from prior assessments.

  3. 03

    Agent execution

    The orchestrator dispatches sub-agents for parallel vulnerability testing. Tools execute in sandboxed containers, findings chain into attack graphs, and coverage tracking ensures nothing is missed.

  4. 04

    Structured output

    JSON and Markdown reports with deduplicated findings, attack graph visualization, reproduction steps, and evidence. Proven payloads promote to muscle memory for future runs.
