How it works

Four-phase cognitive cycle with confidence-driven decisions.

  1. KNOW

    Discovery (0–25% budget)

    Port scan, service detection, tech fingerprinting, endpoint enumeration. Sub-agents run parallel recon at FAST tier. 14 OSINT sources enrich targets.

  2. THINK

    Hypothesis (25–50%)

    Form explicit hypotheses with confidence scores. >75% confidence triggers direct exploitation; <40% gathers more information first. Prioritize high-value targets.

  3. TEST

    Validation (50–80%)

    Test hypotheses with crafted requests from 34K+ payloads. Anti-hallucination pipeline verifies every finding: negative controls, proof-of-execution, confidence scoring, validation judge.

  4. LEARN

    Reporting (80–100%)

    Confirmed findings with CVSS scores, CWE references, and evidence. Quality evaluation across 5 dimensions. Tradecraft memory persists proven techniques for future runs.
